📋 검사 설정

📖 사용 안내

현장 사용 흐름

  1. 대상체에 마커 부착, 셋업 완료
  2. '사진 촬영' 또는 '갤러리에서 선택'
  3. 1~3초 후 결과 표시
  4. ✅이면 본 촬영 진행
  5. ⚠️/❌이면 안내에 따라 셋업 조정 후 재촬영

검사 항목

  • 마커 검출 / ID 매칭
  • 픽셀 크기 (사진 대비 %)
  • 빛 반사 / 과노출
  • 그림자 / 저조도
  • 초점 흐림 / 모션 블러
  • 비스듬한 각도
  • 곡면 변형
